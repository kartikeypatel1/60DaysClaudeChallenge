# 🕵️ Become a Marketing Detective

An interactive React application that teaches marketing analytics through detective-style investigations. Users solve marketing mysteries by collecting evidence, analyzing campaign performance, interpreting customer behavior, and identifying opportunities to improve business outcomes.

---

# 📖 Overview

Successful marketing is built on evidence, not assumptions.

**Become a Marketing Detective** transforms marketing education into an engaging investigative experience where users explore campaign data, uncover hidden insights, connect clues, and solve real-world marketing challenges using analytical thinking.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/your-username/Become-a-Marketing-Detective.git
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
