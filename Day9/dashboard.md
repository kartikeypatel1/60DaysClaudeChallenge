# NutriScope Dashboard Design

## Overview

NutriScope is a modern Nutrition Intelligence Platform designed to help users track calories, macronutrients, micronutrients, and overall nutritional health through an intuitive SaaS-style dashboard.

The interface follows a premium dark theme inspired by modern analytics platforms such as Linear, Stripe Analytics, and Vercel.

---

# Dashboard Features

## 1. Sidebar Navigation

The application uses a fixed vertical sidebar for seamless navigation.

### Sections

#### Overview

* Dashboard

#### Tracking

* Food Log
* Nutrients

#### Insights

* Recommendations
* Meal Planner
* Risk Analysis

#### Settings

* Profile

---

## 2. Today's Dashboard

The dashboard serves as the primary analytics center for nutrition tracking.

### Energy Balance Card

Displays:

* Daily calorie goal completion
* Calories consumed
* Daily target
* Remaining calories
* BMR (Basal Metabolic Rate)

Features:

* Circular progress indicator
* Gradient glow effect
* Real-time updates

---

### Macronutrient Split

Visual representation of:

* Protein
* Carbohydrates
* Fat

Features:

* Doughnut chart
* Percentage breakdown
* Color-coded nutrient categories

---

### Quick Metric Cards

#### Calories

Displays total calories consumed.

#### Protein

Displays total protein intake.

#### Carbs

Displays carbohydrate intake.

#### Fat

Displays fat intake.

Features:

* Large metric values
* Gradient top borders
* Mini trend visualization

---
